# 内容日历规范

## 发布节奏

### AI 日更 (AI日更)
- **频率**: 每个工作日 (周一至周五)
- **时间**: 每日 18:00
- **模板**: `post-ai-news.md`
- **分类**: `AI日更`
- **标签**: `人工智能`, `AI资讯`, `AI日更`

### AI 周报 (AI周报)
- **频率**: 每周五
- **时间**: 每周五 09:00
- **模板**: `post-ai-weekly.md`
- **分类**: `AI周报`
- **标签**: `人工智能`, `AI周报`, `产业动态`, `技术变革`

### 深度分析 (AI大模型/AI应用)
- **频率**: 每月 2-4 篇
- **时间**: 灵活安排
- **模板**: `post-ai-deep-dive.md` 或 `post-ai-tutorial.md`
- **分类**: `AI大模型` 或 `AI应用`
- **标签**: 根据主题添加相关标签

### 智能化标准专题
- **频率**: 按需
- **时间**: 政策发布后 24 小时内
- **模板**: `post-ai-deep-dive.md`
- **分类**: `智能化标准`
- **标签**: `人工智能`, `智能化标准`, `L1-L4`, `AI分级`

---

## 内容分类说明

| 分类 | 说明 | 适用内容 |
|------|------|----------|
| AI大模型 | 大模型相关技术、评测、分析 | 模型评测、技术解读、行业分析 |
| AI应用 | AI工具、应用场景、实践指南 | 工具介绍、使用教程、应用案例 |
| 智能化标准 | AI终端分级、行业标准、政策法规 | 国标解读、政策分析、合规指南 |
| AI周报 | 每周AI领域要闻汇总 | 周度资讯整理 |
| AI日更 | 每日AI快讯 | 当日热点资讯 |
| AI人工智能 | 综合类AI文章 | 不适合以上分类的综合内容 |

---

## 标签使用规范

### 核心标签 (必选)
- `人工智能` - 所有AI相关文章

### 分类标签 (根据内容选择)
- `AI大模型` / `大模型` / `LLM` / `GPT`
- `AI应用` / `AI工具`
- `智能化标准` / `L1-L4` / `AI分级`
- `AI周报` / `AI日更`

### 主题标签 (可选)
- `AI治理` - 监管、政策相关
- `产业动态` - 行业、企业动态
- `技术变革` - 技术突破、创新
- `开源` - 开源相关
- `商业化` - 商业模式、盈利

## 文件命名规范

```
YYYY-MM-DD-{简短描述}-{标识}.md
```

示例:
- `2026-05-08-ai-enterprise-tech-highlights.md`
- `2026-05-08-ai-weekly-newsletter.md`
- `2026-05-08-ai-terminal-standardization-l1-l4.md`

---

## Front-matter 规范

```yaml
---
title: 文章标题
date: 'YYYY-MM-DD HH:MM:SS'
updated: 'YYYY-MM-DD HH:MM:SS'
permalink: /post/{slug}.html
comments: true
toc: true
category:
  - 主分类
  - 副分类(可选)
tags:
  - 标签1
  - 标签2
---
```
