# 文章质量检查清单

**适用范围：** 所有发布到 shenhuanjie.github.io 的文章
**最后更新：** 2026-05-08

---

## 一、标题规范

| 检查项 | 要求 | 状态 |
|--------|------|------|
| 长度 | 中文标题 15-30 字，英文标题 5-10 个单词 | □ |
| 格式 | 不要使用特殊字符（`*`、`#`、`_` 等装饰符） | □ |
| 关键词 | 标题前 10 个字包含核心关键词 | □ |
| 大小写 | 英文单词首字母大写（专有名词除外） | □ |
| 可读性 | 避免生造词、网络用语 | □ |

**示例：**
- ✅ `2026年5月 AI资讯周报（第1周）`
- ✅ `Ollama 本地部署指南：快速搭建 AI 模型服务`
- ❌ `🔥震惊！AI圈发生大事❗`

---

## 二、摘要规范

| 检查项 | 要求 | 状态 |
|--------|------|------|
| 字数 | 中文 80-150 字，英文 50-100 词 | □ |
| 格式 | 使用纯文本，避免 HTML 标签 | □ |
| 内容 | 包含文章核心主题、解决什么问题 | □ |
| 关键词 | 自然嵌入 2-3 个核心关键词 | □ |
| 结尾 | 不要使用"点击查看全文"等引导语句 | □ |

---

## 三、正文内容规范

| 检查项 | 要求 | 状态 |
|--------|------|------|
| 结构 | 包含引言、正文、总结三个部分 | □ |
| 段落 | 每段不超过 4 句话，保持简洁 | □ |
| 列表 | 使用有序/无序列表组织信息 | □ |
| 强调 | 关键词加粗，不要全文大量加粗 | □ |

---

## 四、图片规范

| 检查项 | 要求 | 状态 |
|--------|------|------|
| 格式 | 使用 WebP、PNG、JPG 格式 | □ |
| 尺寸 | 宽度不超过 1200px（ Retina 屏可 2400px） | □ |
| 大小 | 单张图片不超过 500KB | □ |
| 命名 | 使用描述性名称（如 `ollama-install-step1.png`） | □ |
| Alt 文本 | 每张图片添加 `alt` 描述（10-30 字） | □ |
| 位置 | 图片居中显示，与上下文相关 | □ |

**图片压缩建议：**
- 使用 [TinyPNG](https://tinypng.com/) 压缩 JPG/PNG
- 使用 [Squoosh](https://squoosh.app/) 压缩 WebP

---

## 五、代码规范

| 检查项 | 要求 | 状态 |
|--------|------|------|
| 语言标记 | 使用 ` ```language ` 标注语言 | □ |
| 缩进 | 统一使用 4 空格或 Tab（保持一致） | □ |
| 行号 | 长代码（>20行）建议添加行号 | □ |
| 注释 | 关键步骤添加中文注释 | □ |
| 复制按钮 | 确保代码块右上角有复制按钮 | □ |
| 超长代码 | 超过 50 行建议拆分成多个代码块 | □ |

**支持的代码语言：**
```
javascript, typescript, python, java, go, rust, bash, shell,
json, yaml, xml, html, css, sql, markdown, dockerfile
```

## 六、SEO 规范

| 检查项 | 要求 | 状态 |
|--------|------|------|
| 关键词 | 标题、摘要、首段包含核心关键词 | □ |
| 标签 | 添加 3-5 个相关标签 | □ |
| 分类 | 选择一个主分类 + 1个副分类 | □ |
| 永久链接 | 使用英文或拼音，简短（<50字符） | □ |
| 描述 | 每篇文章手动设置 meta description | □ |

---

## 七、元信息规范

| 检查项 | 要求 | 状态 |
|--------|------|------|
| front-matter | 包含 title、date、tags、categories | □ |
| 日期格式 | 使用 `YYYY-MM-DD` 格式 | □ |
| 作者 | 与 config.yml 中的 author 一致 | □ |
| 封面图 | AI/科技类文章建议添加封面图 | □ |

**front-matter 示例：**
```yaml
---
title: Ollama 本地部署指南
date: 2026-05-08
tags:
  - AI工具
  - Ollama
  - 本地部署
categories:
  - AI应用
  - 教程
cover: /images/cover-ollama.png
---
```

## 九、发布前自检清单

发布文章前，请确认以下所有项目已完成：

### 基础检查
- [ ] 标题符合规范
- [ ] 摘要简洁有力
- [ ] 正文结构清晰
- [ ] 代码块可正常显示
- [ ] 图片已正确加载

### SEO 检查
- [ ] 标题包含关键词
- [ ] 标签已添加
- [ ] 分类已选择
- [ ] 永久链接已设置

### 技术检查
- [ ] 本地 `npm run server` 预览正常
- [ ] 页面无控制台错误
- [ ] 移动端显示正常

## 十、常见问题

| 问题 | 解决方案 |
|------|----------|
| 代码块不显示高亮 | 检查语言标记是否正确 |
| 图片无法加载 | 检查路径是否使用绝对路径 |
| 目录不生成 | 检查是否超过 1500 字 |
| 标签不显示 | 检查 front-matter 格式 |
