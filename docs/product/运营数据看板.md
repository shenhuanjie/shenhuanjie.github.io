# 运营数据看板

**文档版本：** v2.2 Phase 3
**更新日期：** 2026-05-08
**适用对象：** 运营人员、产品经理

---

## 一、概述

静态博客项目受限于技术架构，无法实现自动化的实时数据看板。本文档提供手动创建和查看数据看板的方案，以及推荐的看板结构。

### 1.1 核心追踪指标（v2.2 Phase 3 新增）

| 指标类型 | 指标名称 | 计算方式 | 目标值 |
|----------|----------|----------|--------|
| **流量指标** | | | |
| | PV（页面浏览量） | 每次页面访问计 +1 | 日均 ≥500 |
| | UV（独立访客） | 通过 Cookie 识别 | 日均 ≥200 |
| | PV/UV 比 | PV / UV | 2.5-4.0 |
| **用户行为指标** | | | |
| | 平均阅读时长 | 页面停留时间 / 会话数 | ≥2 分钟 |
| | 跳出率 | 单页会话 / 总会话 | ≤60% |
| | 阅读完成率 | article_read_complete 事件数 / PV | ≥30% |
| | 分享率 | share_click 事件数 / PV | ≥1% |
| **转化指标** | | | |
| | 外链点击率 | external_link_click / PV | 5-15% |
| | 目录导航使用率 | toc_navigate / article_pv | 10-30% |

```
├─────────────────────────────────────────────────────────┤

│  ├── 文章发布统计                                        │

│  └── 转化漏斗分析                                        │
└─────────────────────────────────────────────────────────┘
```

### 2.2 指标定义与计算

#### 流量指标

| 指标 | 定义 | 数据来源 | 更新频率 |
|------|------|----------|----------|
| PV | 页面被访问的总次数 | GA4 screenPageViews | 实时 |
| UV | 一定周期内独立访客数 | GA4 totalUsers | 实时 |
| PV/UV 比 | 人均页面浏览数 | PV / UV | 每日 |

#### 用户行为指标

| 指标 | 定义 | 计算公式 | 理想范围 |
|------|------|----------|----------|
| 平均阅读时长 | 用户在站点平均停留时间 | totalEngagementTime / sessions | 2-5 分钟 |
| 跳出率 | 只访问一个页面就离开的会话占比 | bounceRate | < 60% |
| 阅读完成率 | 滚动超过 90% 的文章占比 | article_read_complete / pageviews | > 30% |
| 分享率 | 触发分享点击的文章占比 | share_click / pageviews | > 1% |

#### 转化漏斗指标

```
访问 → 浏览文章 → 阅读完成 → 分享/外链 → 转化
```

| 漏斗阶段 | 指标 | 计算 |
|----------|------|------|
| 进入 | 会话数 | sessions |
| 浏览文章 | 文章页会话 | sessions with article page |
| 阅读完成 | 完成阅读数 | article_read_complete events |
| 分享/外链 | 互动数 | share_click + external_link_click |
| 转化 | 目标达成 | (待定义，如订阅) |

## 三、GA4 数据看板配置

### 3.1 创建自定义仪表盘

1. 登录 [Google Analytics](https://analytics.google.com/)
2. 进入「报告」→「库」
3. 点击「新建自定义仪表盘」

### 3.2 推荐组件配置

#### 核心指标卡片

| 指标 | 指标类型 | 显示格式 |
|------|----------|----------|
| 活跃用户 | `totalUsers` | 数字 + 趋势箭头 |
| 会话 | `sessions` | 数字 |
| 页面浏览量 | `screenPageViews` | 数字 |
| 平均参与时长 | `averageSessionDuration` | 分:秒 格式 |
| 跳出率 | `bounceRate` | 百分比 |
| 阅读完成数 | `article_read_complete` (自定义事件) | 数字 |
| 分享点击数 | `share_click` (自定义事件) | 数字 |

#### 趋势图组件

```
报告类型：时序图
维度：日期
指标：页面浏览量、用户数、会话数、平均参与时长
时间段：最近 30 天
```

#### TOP 10 文章列表

```
报告类型：数据表
维度：页面路径
指标：页面浏览量、平均参与时长、跳出率、article_read_complete
排序：页面浏览量（降序）
过滤：页面路径包含 /20XX/
```

#### 流量来源分布

```
报告类型：饼图/ donut
维度：会话来源
指标：用户数、会话数
时间段：最近 30 天
```

### 3.3 自定义事件报告配置

在 GA4 中查看自定义事件：

1. 报告 → 参与度 → 事件
2. 找到自定义事件：`article_read_complete`, `share_click`, `external_link_click`
3. 点击事件名称查看详细参数

### 3.4 分享仪表盘

1. 在仪表盘右上角点击「分享」
2. 选择「发布到生态系统」或「导出为 PDF」
3. 设置访问权限

## 六、自定义报告模板

### 6.1 周报数据提取路径

#### 1. 流量概览

- 路径：报告 → 概览
- 提取指标：用户数、页面浏览量、会话数、平均参与时长、跳出率

#### 2. 流量来源

- 路径：报告 → 用户获取 → 流量获取
- 提取维度：会话来源、媒介
- 提取指标：用户数、转化数

#### 3. 内容表现

- 路径：报告 → 参与度 → 页面和屏幕
- 提取维度：页面路径
- 提取指标：页面浏览量、平均参与时长、跳出率

#### 4. 事件统计

- 路径：报告 → 参与度 → 事件
- 提取事件：`article_read_complete`, `share_click`, `external_link_click`, `search_query`

### 6.2 数据导出方法

1. 在 GA4 报告中，调整时间范围
2. 点击「导出」按钮
3. 选择格式（CSV/Google 表格/Excel）
4. 保存到本地或 Google Drive

## 七、数据质量监控

### 7.1 异常告警配置

在 GA4 中设置自定义告警：

1. 进入「管理」→「自定义告警」
2. 创建告警规则：

| 告警名称 | 条件 | 阈值 |
|----------|------|------|
| 流量骤降 | 用户数环比 | < -30% |
| 流量暴涨 | 用户数环比 | > +50% |
| 跳出率异常 | 跳出率 | > 80% |
| PV 异常 | 页面浏览量 | < 100/天 |
| 阅读完成率低 | article_read_complete 率 | < 10% |

### 7.2 告警通知设置

- 通知频率：每日一次
- 通知方式：邮件
- 接收人：运营人员邮箱

## 附录 A：核心指标计算公式

```
阅读完成率 = article_read_complete 事件数 / 文章页 PV × 100%
分享率 = share_click 事件数 / 总 PV × 100%
外链点击率 = external_link_click 事件数 / 总 PV × 100%
人均页面数 = 总 PV / 总 UV
跳出率 = 单页会话数 / 总会话数 × 100%
```
