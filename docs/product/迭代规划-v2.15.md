## 三、任务详情

### 3.1 关键词库建设

**目标：** 建立科学的关键词分类体系

```yaml
# 关键词分类
流量词:
  - AI大模型
  - ChatGPT
  - LLM
  - 大模型

长尾词:
  - "2026年AI Agent开发指南"
  - "RAG技术原理详解"
  - "AI编程工具横向对比"

品牌词:
  - 程序猿视界
  - 技术博客
```

### 3.2 内部链接策略

**目标：** 构建文章间语义链接网络

```markdown
<!-- 链接规范 -->
1. 同系列文章：建立「上/下篇」导航
2. 相关主题文章：在文末「相关阅读」添加 3-5 篇
3. 概念引用：在首次出现术语时链接到解释文章
4. 分类页面：在文末添加「更多XX类文章」
```

### 3.3 发布检查清单自动化

**目标：** 自动化文章发布前的质量检查

```javascript
// pre-publish-check.js 检查项
const CHECKLIST = [
  { name: 'title_length', rule: title => title.length >= 10 && title.length <= 60 },
  { name: 'description_exists', rule: post => post.description && post.description.length >= 50 },
  { name: 'keywords_optimized', rule: post => post.keywords && post.keywords.length <= 5 },
  { name: 'has_cover_image', rule: post => post.cover_image !== null },
  { name: 'code_blocks_formatted', rule: post => /* 代码块语法检查 */ },
  { name: 'internal_links', rule: post => /* 至少3个内部链接 */ },
  { name: 'external_links_safe', rule: post => /* 外链格式正确 */ },
  { name: 'reading_time_reasonable', rule: post => post.reading_time >= 3 && post.reading_time <= 20 }
];
```

### 3.4 GA4 漏斗深度分析

**目标：** 理解用户行为路径

```
用户路径漏斗：
1. 搜索流量 → 落地页
2. 落地页 → 内链跳转
3. 内链跳转 → 深度阅读（>3分钟）
4. 深度阅读 → 互动（评论/分享/收藏）
5. 互动 → 回访（次日/周）

优化策略：
- 高跳出率页面：优化开头内容，增加内部链接
- 低互动页面：增加行动号召（CTA）
- 高价值页面：增加曝光和入口
```
