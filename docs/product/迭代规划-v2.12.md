## 四、任务详情

### 4.1 代码质量审计

**目标：** 建立代码规范，清理无用代码

```bash
# ESLint 检查
npm run lint

# Prettier 格式化
npm run format

# 死代码检测
find source/_posts -name "*.md" -empty
```

**文件位置：** `package.json`、`.eslintrc.js`、`.prettierrc`

### 4.2 图片加载优化

**目标：** 提升页面加载速度

```javascript
// 懒加载实现
<img src="placeholder.png" data-src="actual-image.jpg" class="lazyload" />

// WebP 支持检测
if (supportsWebP()) {
  useWebPImages();
}
```

### 4.3 Meta 标签规范化

**目标：** 全站 SEO 标签统一

```yaml
# _config.yml 检查项
og_image: /images/og-default.jpg
twitter_card: summary_large_image
```
