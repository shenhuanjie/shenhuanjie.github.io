**博客名称：** 程序猿视界 (shenhuanjie.github.io)

**版本：** v2.16

## 一、设计原则

### 1.1 核心原则

1. **用户体验优先**
   - 广告不得影响正常阅读体验
   - 不遮挡主要内容
   - 加载速度影响最小化

2. **收益最大化**
   - 选择高可见度位置
   - 合理控制广告密度
   - 适配多种设备

3. **可量化评估**
   - 每个广告位独立追踪
   - A/B 测试支持
   - 数据驱动优化

## 三、PC端广告位详细设计

### 3.1 AD-01：右侧边栏顶部（主广告位）

**位置示意：**
```
+---------------------------+
|        页面主体           |
|  +---------------------+  |
|  |                     |  |
|  |    文章列表/正文     |  |
|  |                     |  |
|  +---------------------+  |
|                    +-----+|
|                    | AD-01|| ← 300x250
|                    +-----+|
|                    | AD-02|| ← 300x250
|                    +-----+|
+---------------------------+
```

**规格参数：**
- 尺寸：300x250（中矩形广告）
- 位置：右侧固定，页面滚动时固定在视口
- 展示规则：首页、分类页、标签页
- 隐藏规则：文章详情页（避免干扰阅读）
- 最多展示：1 个

**技术实现：**
```html
<div class="sidebar-ad" data-ad-position="AD-01">
  <!-- 广告代码 -->
</div>
```

### 3.2 AD-02：右侧边栏底部（辅助广告位）

**规格参数：**
- 尺寸：300x250 或 160x600（长方形）
- 位置：右侧底部，页面滚动时固定
- 展示规则：首页、分类页、标签页
- 隐藏规则：文章详情页
- 最多展示：1 个

### 3.3 AD-03：文章内广告位

**位置：**
- 插入位置：文章正文第二段之后、第三段之前
- 如果文章少于 3 段，则不显示

**示意图：**
```
第一段内容...
第二段内容...

+------------------------+
|      [AD-03 文字链]    |
+------------------------+

第三段内容...
...（后续正文）
```

**规格参数：**
- 形式：文字链接广告（不干扰阅读流）
- 展示规则：仅文章页
- 最多显示：1 个
- 样式：与正文文字颜色一致，底部添加"广告"标识

**技术实现：**
```html
<article class="post-content">
  <p>第一段...</p>
  <p>第二段...</p>
  <div class="in-article-ad" data-ad-position="AD-03">
    <!-- 文字链广告 -->
  </div>
  <p>第三段...</p>
</article>
```

### 3.4 AD-04：文章底部广告位

**位置：**
- 位置：文章正文结束后、相关文章之前
- 或者：评论区之前

**规格参数：**
- 尺寸：响应式（728x90 或自适应）
- 展示规则：仅文章页
- 最多展示：1 个
- 样式：简洁边框，与页面风格统一

## 四、移动端广告位设计

### 4.1 响应式策略

移动端屏幕宽度有限，广告位需要重新规划：
- 右侧边栏广告在移动端隐藏
- 文章内广告保留但尺寸调整
- 新增底部固定广告位

### 4.2 AD-05：移动端底部广告

**位置：**
- 固定在屏幕底部
- 距底部 0px，高度 50-100px

**规格参数：**
- 尺寸：320x100（移动端大图广告）
- 位置：屏幕底部固定
- 展示规则：全站移动端
- 隐藏条件：用户向下滚动时自动隐藏（滑至顶部后显示）
- 频率控制：24小时只展示1次

**技术实现：**
```html
<div class="mobile-bottom-ad" data-ad-position="AD-05" style="display:none;">
  <button class="close-btn">×</button>
  <!-- 广告内容 -->
</div>
```

### 4.3 移动端广告位禁用规则

以下 PC 端广告位在移动端禁用：
- AD-01 右侧边栏顶部 → 移动端隐藏
- AD-02 右侧边栏底部 → 移动端隐藏
- AD-03 文章内广告 → 保留但缩小尺寸

## 五、广告加载策略

### 5.1 懒加载机制

```javascript
// 广告懒加载
const loadAdWhenVisible = (adElement) => {
  const observer = new IntersectionObserver((entries) => {
    entries.forEach(entry => {
      if (entry.isIntersecting) {
        loadAd(adElement);
        observer.unobserve(adElement);
      }
    });
  });
  observer.observe(adElement);
};
```

### 5.2 异步加载

所有广告脚本使用异步加载模式：
```html
<script async src="..."></script>
```

### 5.3 占位符策略

在广告加载前显示占位符，防止页面布局跳动：
```css
.ad-placeholder {
  background: #f5f5f5;
  min-height: 250px;
  display: flex;
  align-items: center;
  justify-content: center;
}
```

## 七、频率控制

### 7.1 弹窗广告控制

| 类型 | 频率限制 |
|------|----------|
| 全屏弹窗 | 24小时最多1次 |
| 关注公众号弹窗 | 24小时最多1次 |
| 底部固定广告 | 始终显示，可关闭 |

### 7.2 冷却机制

```javascript
const COOLDOWN_KEY = 'ad_shown_timestamp';
const COOLDOWN_DURATION = 24 * 60 * 60 * 1000; // 24小时

const shouldShowAd = () => {
  const lastShown = localStorage.getItem(COOLDOWN_KEY);
  if (!lastShown) return true;
  return Date.now() - parseInt(lastShown) > COOLDOWN_DURATION;
};
```

## 八、数据追踪

### 8.1 追踪指标

| 指标 | 说明 | 工具 |
|------|------|------|
| 展示量 | 广告展示次数 | GA4 / 广告平台后台 |
| 点击量 | 广告点击次数 | GA4 / 广告平台后台 |
| CTR | 点击率 = 点击/展示 | 计算得出 |
| 收益 | 广告收益 | 广告平台后台 |
| RPM | 千次展示收益 | 计算得出 |

### 8.2 事件追踪

```javascript
// 广告展示追踪
document.querySelectorAll('.ad-container').forEach(ad => {
  ad.addEventListener('ad:impression', (e) => {
    gtag('event', 'ad_impression', {
      'ad_position': e.target.dataset.adPosition
    });
  });
});
```
