## 四、任务详情

### 4.1 英文文章体系建设

**目标：** 建立可持续的英文内容生产流程

```yaml
# 英文文章 frontmatter 模板
---
title: "Article Title in English"
description: "English description for SEO"
categories: [AI-LLM]
tags: [LLM, AI]
lang: en
translation_of: 2026-05-09-original-article-slug
---

# 英文文章规范
1. 文章标题翻译
2. description 需要重新撰写（不能机翻）
3. 保持代码示例原样
4. 添加英文目录（Table of Contents）
```

**文件位置：** `source/_posts/en/`

### 4.2 LCP 优化方案

**目标：** 将 LCP 时间降低至 2.5s 以内

```html
<!-- 关键图片预加载 -->
<link rel="preload" as="image" href="/images/hero.jpg">

<!-- 字体预加载 -->
<link rel="preload" as="font" href="/fonts/iconfont.woff2">

<!-- 关键 CSS 内联 -->
<style>
  /* Above-the-fold 样式 */
</style>
```

**优化措施：**
1. 封面图 WebP 格式 + CDN
2. 字体文件 woff2 压缩
3. CSS 关键路径内联
4. 图片懒加载（非首屏）

### 4.3 用户漏斗分析

**目标：** 建立数据驱动的增长模型

```javascript
// 漏斗事件追踪
gtag('event', 'funnel_step', {
  step: 'homepage_view',
  step_number: 1
});

gtag('event', 'funnel_step', {
  step: 'article_click',
  step_number: 2
});

gtag('event', 'funnel_step', {
  step: 'article_read_complete',
  step_number: 3
});
```

**漏斗阶段：**
1. 首页浏览 → 2. 文章点击 → 3. 文章阅读 → 4. 深度互动
