### 5.2 分享海报生成

使用 canvas 生成带标题、封面图、二维码的分享图：

```javascript
// 分享海报生成方案
async function generateShareImage(article) {
  const canvas = createCanvas(600, 800);
  const ctx = canvas.getContext('2d');

  // 绘制背景
  ctx.fillStyle = '#ffffff';
  ctx.fillRect(0, 0, 600, 800);

  // 绘制封面图
  ctx.drawImage(article.cover, 0, 0, 600, 400);

  // 绘制标题
  ctx.fillStyle = '#333333';
  ctx.font = 'bold 28px sans-serif';
  ctx.fillText(article.title, 40, 480, 520);

  // 绘制二维码
  const qr = await QRCode.toDataURL(article.url);
  ctx.drawImage(qr, 450, 650, 120, 120);

  return canvas.toDataURL('image/png');
}
```

## 六、CI/CD 质量门禁详情

### 6.1 GitHub Actions Workflow

```yaml
# .github/workflows/quality-check.yml
name: Content Quality Check
on:
  push:
    branches: [main]
  pull_request:
    branches: [main]

jobs:
  quality-check:
    runs-on: ubuntu-latest
    steps:
      - uses: actions/checkout@v4

      - name: Setup Node
        uses: actions/setup-node@v4
        with:
          node-version: '20'

      - name: Install dependencies
        run: npm ci

      - name: Validate frontmatter
        run: node scripts/validate-frontmatter.js

      - name: Check article links
        run: node scripts/check-article-links.js

      - name: Check article images
        run: node scripts/check-article-images.js

      - name: Quality gate
        if: failure()
        run: exit 1
```

### 6.2 检查脚本要求

| 脚本 | 检查内容 | 失败条件 |
|------|----------|----------|
| validate-frontmatter.js | title, date, description, categories, tags | 必填字段缺失 |
| check-article-links.js | 内部链接 + 外部链接 | 死链、404 |
| check-article-images.js | 图片路径、CDN 可用性 | 图片不存在 |

## 七、发布通知自动化详情

### 7.1 飞书通知

```javascript
// 飞书 Webhook 通知
async function sendFeishuNotification(article) {
  const webhook = process.env.FEISHU_WEBHOOK_URL;
  const message = {
    msg_type: 'interactive',
    card: {
      header: {
        title: { tag: 'plain_text', content: '📝 新文章发布' },
        template: 'green'
      },
      elements: [{
        tag: 'div',
        text: {
          tag: 'lark_md',
          content: `**${article.title}**\n\n${article.description}`
        }
      }]
    }
  };

  await fetch(webhook, {
    method: 'POST',
    headers: { 'Content-Type': 'application/json' },
    body: JSON.stringify(message)
  });
}
```

### 7.2 通知触发时机

| 事件 | 触发条件 | 通知渠道 |
|------|----------|----------|
| 新文章发布 | 文章发布到 public | 飞书 |
| 定时任务完成 | 采集脚本执行完毕 | 飞书 |
| 质量检查失败 | CI/CD 检查不通过 | 钉钉 |
