## 四、任务详情

### 4.1 hreflang 标签验证

**目标：** 确认多语言页面被搜索引擎正确识别

```html
<!-- 验证点：英文页面 head.ejs -->
<link rel="alternate" hreflang="en" href="https://shenhuanjie.github.io/en/...">
<link rel="alternate" hreflang="zh-CN" href="https://shenhuanjie.github.io/...">
<link rel="alternate" hreflang="x-default" href="https://shenhuanjie.github.io/...">
```

**验证方法：**
1. 检查英文页面是否生成正确 hreflang
2. 使用 Google Search Console 验证多语言覆盖
3. 使用 `site:shenhuanjie.github.io/en` 验证收录

### 4.2 微信公众号内容适配

**目标：** 建立适合微信生态的内容格式

```yaml
# 微信公众号适配规范
---
title: "公众号标题（可与原文不同，更吸引眼球）"
description: "公众号导语，1-2句话概括文章价值"
cover_image: "竖版封面图 2.35:1 (900x383)"
content_format: "图文"
diversion_text: "关注『程序猿视界』公众号，回复『AI』获取更多资讯"
---
```

**分发流程：**
1. 源文章发布
2. 自动生成公众号适配版本
3. 手动调整后发布
4. 原文添加「转载声明」和「公众号引流」

### 4.3 评论系统优化

**目标：** 降低评论门槛，提升评论量

**优化方案：**
1. 简化评论表单（可选昵称邮箱）
2. 增加「赞同/反对」快速反应
3. 添加「收藏文章」快捷按钮
4. 显示评论数量预览

```javascript
// 快速反应事件
gtag('event', 'article_reaction', {
  reaction_type: 'like', // like, disagree, bookmark
  page_title: pageInfo.title,
  page_url: pageInfo.url
});
```

### 4.4 LCP 优化实施

**目标：** 将 LCP 时间降低至 2.5s 以内

**优化措施：**
1. 封面图 WebP 格式 + CDN
2. 字体文件 woff2 压缩 + 预加载
3. CSS 关键路径内联
4. 图片懒加载（非首屏）

```html
<!-- 关键图片预加载 -->
<link rel="preload" as="image" href="/images/cover.webp">

<!-- 字体预加载 -->
<link rel="preload" as="font" href="/fonts/iconfont.woff2">
```
