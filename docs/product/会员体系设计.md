# 付费内容会员体系设计

**博客名称：** 程序猿视界 (shenhuanjie.github.io)
**设计日期：** 2026-05-11
**版本：** v2.16

## 一、设计背景

### 1.1 目的

为技术博客建立可持续的付费内容商业模式，通过提供高质量深度内容吸引用户付费，同时保持免费内容生态的健康发展。

### 1.2 设计原则

1. **免费优先** - 保持 80% 以上内容免费，维护 SEO 和用户基础
2. **价值付费** - 付费内容必须提供实质性深度价值
3. **渐进式付费** - 从打赏到会员，阶梯式商业化
4. **技术人群友好** - 定价合理，不割韭菜

## 四、付费墙策略

### 4.1 软付费墙（推荐）

**实现方式：**
- 文章前 50% 内容免费阅读
- 后 50% 内容需要登录/付费
- 显示"登录后继续阅读"提示

**示意图：**
```
+----------------------------------+
|          文章标题                 |
|                                  |
| [免费阅读部分]                    |
| 第一段...                         |
| 第二段...                         |
| 第三段...                         |
|                                  |
| ======== 付费墙 ========          |
|                                  |
| [登录后继续阅读]                  |
| - - - - - - - - - - -            |
| 剩余内容（付费可见）              |
+----------------------------------+
```

### 4.2 硬付费墙

**适用场景：**
- 高价值付费专栏
- 系列课程完整内容
- 商业级技术方案

### 4.3 付费墙触发规则

| 条件 | 触发付费墙 |
|------|------------|
| 未登录用户 | 显示登录/注册提示 |
| 免费用户 | 显示升级会员提示 |
| 付费会员 | 正常阅读 |

### 5.2 定价设计

- 价格：19 元/月
- 对比：相当于一杯咖啡
- 权益：付费内容 + 技术问答

#### 年度会员
- 价格：169 元/年（约 14 元/月）
- 折扣：相当于免费 2.6 个月
- 权益：月度会员全部 + PRO 社群

#### 年度 PRO
- 价格：299 元/年（约 25 元/月）
- 折扣：相比月度 PRO 便宜 47%
- 权益：全部权益 + 优先咨询

### 5.3 首单优惠策略

| 优惠类型 | 优惠幅度 | 有效期 |

## 七、技术实现方案

### 7.1 付费墙前端实现

```javascript
// 付费墙组件
const Paywall = ({ children, isPaid }) => {
  if (isPaid) {
    return <>{children}</>;
  }

  return (
    <div className="paywall">
      <div className="free-preview">
        {/* 免费预览部分 */}
      </div>
      <div className="paywall-overlay">
        <h3>登录后继续阅读</h3>
        <p>开通会员解锁全部付费内容</p>
        <div className="price-tag">19元/月起</div>
        <button className="btn-upgrade">立即升级</button>
      </div>
    </div>
  );
};
```

### 7.2 后端权限控制

```javascript
// 中间件：检查内容访问权限
const checkContentAccess = async (req, res, next) => {
  const { contentId, userId } = req;

  const content = await getContent(contentId);

  if (!content.isPremium) {
    return next(); // 免费内容
  }

  const user = await getUser(userId);
  if (user.memberLevel >= 1) {
    return next(); // 付费会员
  }

  return res.status(403).json({
    error: 'PREMIUM_CONTENT',
    message: '请升级会员继续阅读'
  });
};
```

### 7.3 内容加密方案

```javascript
// 付费内容加密存储
const encryptContent = (content) => {
  return crypto.aesEncrypt(content, process.env.CONTENT_KEY);
};

// 付费内容解密（仅付费用户）
const decryptContent = (encrypted, userLevel) => {
  if (userLevel < 1) {
    throw new Error('Unauthorized access');
  }
  return crypto.aesDecrypt(encrypted, process.env.CONTENT_KEY);
};
```

## 八、用户引导流程

### 8.1 免费转付费流程

```
用户访问免费文章
      ↓
阅读过程中看到付费标识
      ↓
点击付费内容（提示登录）
      ↓
注册/登录账号
      ↓
显示付费墙和升级选项
      ↓
选择套餐 → 支付 → 成为会员
      ↓
解锁全部付费内容
```

### 8.2 会员升级引导

| 场景 | 引导文案 |
|------|----------|
| 免费用户点击付费内容 | "这篇是付费内容，开通会员即可阅读" |
| 免费用户阅读付费文章 | "升级会员，月费仅 19 元，畅读全部付费内容" |
| 年度会员到期前 | "您的会员即将到期，续费可享 8 折优惠" |
| 高活跃免费用户 | "您已阅读 20 篇付费内容，开通会员更划算" |
