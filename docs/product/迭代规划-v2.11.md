## 三、任务详情

### 3.1 GA4 事件追踪基础版

**目标：** 实现文章阅读、分享的基础事件追踪

```javascript
// 基础事件追踪代码
// 放置在文章页模板中

// 文章阅读开始
gtag('event', 'article_view', {
  article_id: '{{ post.title }}',
  article_category: '{{ post.categories }}',
  author: '{{ post.author }}'
});

// 滚动深度追踪（滚动 > 70%）
$(window).scroll(function() {
  const scrollPercent = Math.round(($(window).scrollTop() / $(document).height()) * 100);
  if (scrollPercent > 70 && !window.scrollTracked) {
    window.scrollTracked = true;
    gtag('event', 'article_read', {
      article_id: '{{ post.title }}',
      read_depth: scrollPercent
    });
  }
});

// 分享按钮点击
$('.share-btn').on('click', function() {
  const platform = $(this).data('platform');
  gtag('event', 'share', {
    method: platform,
    article_id: '{{ post.title }}'
  });
});
```

**文件位置：** `themes/Chic/layout/_partial/article.ejs` 或 `themes/Chic/scripts/`

### 3.2 搜索建议功能

**目标：** 实现实时搜索补全

**技术方案：** 基于 Algolia DocSearch 或自建搜索索引

```javascript
// 搜索建议实现
$('#search-input').on('input', debounce(async function() {
  const query = $(this).val();
  if (query.length < 2) return;

  // 搜索建议 API
  const suggestions = await fetch(`/api/search/suggest?q=${encodeURIComponent(query)}`);
  showSuggestions(suggestions);
}, 300));
```

**搜索建议数据源：**
1. 文章标题
2. 标签云
3. 热门搜索词（GA4 数据）

### 3.3 评论审核机制

**目标：** 自动化评论审核

**功能优先级：**
1. 敏感词过滤（关键词库 + 正则）

3. 审核通知（飞书 Webhook）
