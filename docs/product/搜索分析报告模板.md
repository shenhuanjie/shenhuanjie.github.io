## 一、数据采集格式

### 1.1 数据来源

搜索分析数据通过 `localStorage` 中的 `blog_search_analytics` 键存储。

### 1.2 数据结构

```javascript
{
  "searches": [
    {
      "keyword": "AI大模型",
      "resultCount": 12,
      "timestamp": 1746758400000
    }
  ],
  "noResults": [
    {
      "keyword": "Vue4",
      "timestamp": 1746758400000
    }
  ],
  "clickData": [
    {
      "keyword": "AI大模型",
      "articleUrl": "/post/xxx.html",
      "position": 1,
      "timestamp": 1746758400000
    }
  ]
}
```

### 1.3 关键指标

| 指标 | 说明 | 用途 |
|------|------|------|
| totalSearches | 总搜索次数 | 衡量搜索功能使用率 |
| uniqueKeywords | 独立关键词数 | 反映内容覆盖面 |
| noResultCount | 无结果搜索次数 | 发现内容缺口 |
| clickCount | 搜索后点击次数 | 衡量搜索结果相关性 |

## 四、数据导出方法

### 4.1 前端导出

在浏览器控制台执行：

```javascript
// 获取搜索数据
const data = window.searchAnalytics.exportSearchData();
console.log(JSON.stringify(data, null, 2));

// 下载为 JSON 文件
const blob = new Blob([JSON.stringify(data, null, 2)], { type: 'application/json' });
const url = URL.createObjectURL(blob);
const a = document.createElement('a');
a.href = url;
a.download = `search-analytics-${new Date().toISOString().split('T')[0]}.json`;
a.click();
```

### 4.2 数据字段说明

| 字段 | 类型 | 说明 |
|------|------|------|
| summary | Object | 搜索统计摘要 |
| popularSearches | Array | 热门搜索列表 |
| noResultSearches | Array | 无结果搜索列表 |
| recentSearches | Array | 最近搜索历史 |
| exportedAt | String | 导出时间 ISO 格式 |
