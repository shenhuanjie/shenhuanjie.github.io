## 一、核心指标定义

### 1.1 用户获取指标

| 指标名称 | 定义 | 计算方式 | 当前目标 |
|----------|------|----------|----------|
| 会话数 | 用户访问会话总数 | GA4 自动追踪 | 持续增长 |
| 新访客数 | 首次访问用户数 | `new_visitor` 事件计数 | 占总会话 60%+ |
| 回访客数 | 7日内再次访问用户 | `return_visitor` 事件计数 | 占总会话 40%+ |
| 跳出率 | 单页会话 / 总会话 | (单页会话 / 总会话) × 100% | < 50% |
| 页面浏览量 | PV | `page_view` 事件计数 | 持续增长 |

### 1.2 用户互动指标

| 指标名称 | 定义 | 计算方式 | 当前目标 |
|----------|------|----------|----------|
| 平均阅读时长 | 用户在页面的平均停留时间 | `reading_time` 事件均值 | > 2 分钟 |
| 滚动深度 | 用户滚动百分比 | `scroll_depth` 事件均值 | > 70% |
| 深度阅读率 | 滚动 > 50% 的会话比例 | scroll_depth >= 50 会话 / 总会话 | > 40% |
| 分享次数 | 分享按钮点击次数 | `share_article` 事件计数 | > 2% 分享率 |
| 评论数 | 评论提交次数 | `comment_submit` 事件计数 | > 1% 评论率 |
| 收藏数 | 收藏操作次数 | `favorite_article` 事件计数 | > 3% 收藏率 |
| 搜索次数 | 站内搜索次数 | `search` 事件计数 | - |

### 1.3 转化指标

| 指标名称 | 定义 | 计算方式 | 当前目标 |
|----------|------|----------|----------|
| 7日留存率 | 7日内回访用户 / 新访客 | return_visitor / new_visitor | > 15% |
| 30日留存率 | 30日内回访用户 / 新访客 | 30日回访 / 新访客 | > 8% |
| 转化数 | 最终转化行为次数 | `conversion` 事件计数 | - |
| 转化率 | 转化数 / 会话数 | conversion / session | > 1% |

## 二、数据采集方案

### 2.1 事件追踪架构

```
┌─────────────────────────────────────────────────────────────┐
│                    user-interaction.js                       │
├─────────────────────────────────────────────────────────────┤
│  ┌─────────────┐  ┌─────────────┐  ┌─────────────────────┐  │
│  │ScrollTracker│  │ReadingTime │  │ReturnVisitorTracker │  │
│  │             │  │ Tracker    │  │                     │  │
│  └─────────────┘  └─────────────┘  └─────────────────────┘  │
├─────────────────────────────────────────────────────────────┤
│              事件发送层 (gtag)                               │
│  - scroll_depth     - reading_time      - return_visitor    │
│  - share_article    - comment_submit    - favorite_article   │
│  - search           - new_visitor       - conversion       │
└─────────────────────────────────────────────────────────────┘
                              ↓
                    ┌─────────────────┐
                    │   GA4 服务器    │
                    └─────────────────┘
```

### 2.2 事件参数定义

| 事件名称 | 触发条件 | 参数 |
|----------|----------|------|
| `scroll_depth` | 滚动超过 50%/75%/100% | depth_percent, page_title, page_url |
| `reading_time` | 页面离开或每30秒 | reading_seconds, max_scroll_depth, is_page_leave |
| `return_visitor` | 回访用户访问 | days_since_last_visit, page_title, page_url |
| `new_visitor` | 首次访问 | visit_count |
| `share_article` | 分享按钮点击 | method, page_title, page_url |
| `comment_submit` | 评论提交 | action, page_title, page_url |
| `favorite_article` | 收藏操作 | action, page_title, page_url |
| `search` | 站内搜索 | search_term, result_count |

### 2.3 自定义维度

在 GA4 管理后台配置以下自定义维度：

| 维度名称 | 范围 | 参数 |
|----------|------|------|
| 文章标题 | 用户 | page_title |
| 文章分类 | 用户 | article_category |
| 分享平台 | 事件 | method |
| 阅读深度 | 事件 | depth_percent |
| 回访间隔 | 用户 | days_since_last_visit |
