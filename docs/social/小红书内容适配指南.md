# 小红书内容适配指南

**文档版本：** v1.0
**创建日期：** 2026-05-11
**适用场景：** 将博客文章适配发布至小红书

## 一、平台概述

| 参数 | 数值/说明 |
|------|-----------|
| 用户画像 | 18-35 岁女性为主，年轻化、高消费力 |

## 二、内容格式

### 2.1 短图文格式（推荐）

小红书以短图文为主，博客长文需要进行大幅精简：

```markdown
## 格式结构

【封面标题】（20字以内，制造悬念或承诺）
正文内容（建议 300-500 字）
话题标签（5-10 个）
互动引导（可选）
```

### 2.2 内容转换原则

| 博客内容 | 小红书适配 |
|---------|-----------|
| 3000+ 字长文 | 精简为 300-500 字核心要点 |
| 技术细节 | 转化为「教程/攻略」形式 |
| 深度分析 | 拆分为系列笔记 |
| 代码片段 | 转化为截图或简化说明 |

### 2.3 标题公式

```
# 标题模板

痛点型: 手把手教你[技能]，小白也能[效果]
数字型: [数字]个[主题]技巧，第[N]个绝了
对比型: 终于找到了！[对比]的[替代品]
合集型: [主题]合集 | 第[数字]期 [子主题]
种草型: 吹爆这款[产品]！[效果]太绝了
```

## 三、封面图制作规范

### 3.1 图片尺寸

| 参数 | 数值 |
|------|------|
| 比例 | 3:4（竖版推荐） |
| 宽度 | 1080px |
| 高度 | 1440px |
| 格式 | JPG/PNG |
| 文件大小 | ≤ 5MB |

### 3.2 封面设计要素

```
┌─────────────────────────────────┐
│                                 │
│     [大字标题 24-32px]          │
│                                 │
│     [副标题/描述 16-20px]       │
│                                 │
│         [产品/内容图]           │
│                                 │
│  [Logo]                         │
└─────────────────────────────────┘
```

### 3.3 封面模板推荐

```javascript
// 小红书封面图生成配置
const weiboCoverConfig = {
```

## 四、话题标签策略

### 4.1 标签分类

| 标签类型 | 示例 | 数量建议 |
|---------|------|---------|
| 领域标签 | #AI #人工智能 #科技 | 1-2 个 |
| 场景标签 | #职场 #学习 #生活 | 1-2 个 |
| 人群标签 | #打工人 #学生党 #程序员 | 1 个 |
| 热点标签 | #2026 #热点 #趋势 | 1-2 个 |
| 平台标签 | #小红书 #薯条小助手 | 1 个 |

### 4.2 标签使用技巧

```javascript
// 生成小红书标签
function generateWeiboTags(article) {
  const tags = [];

  // 1. 固定标签（必加）
  tags.push('#程序猿视界');  // 品牌标签
  tags.push('#科技');        // 领域标签

  // 2. 从文章标签转换
  if (article.tags) {
    article.tags.slice(0, 3).forEach(tag => {
      tags.push(`#${tag.replace(/\s+/g, '')}`);
    });
  }

  // 3. 从内容提取热点词
  const hotTopics = extractHotTopics(article.content);
  tags.push(...hotTopics.slice(0, 2));

  // 4. 限制数量
  return tags.slice(0, 10);
}
```

### 4.3 禁用标签

- 敏感政治话题相关
- 竞品负面评价
- 虚假夸大宣传

### 5.2 内容技巧

| 技巧 | 说明 |
|------|------|
| emoji 适度 | 每段 1-2 个，增加活力 |
| 分段清晰 | 每段不超过 5 行 |
| 金句加粗 | 核心观点用 **加粗** 强调 |
| 留白适当 | 段间空行，方便阅读 |

## 六、图片制作建议

### 6.1 图片数量

| 内容类型 | 图片数量 |
|---------|---------|
| 攻略教程 | 3-6 张（封面 + 步骤图 + 结尾图） |
| 种草推荐 | 4-9 张（多角度/对比图） |
| 合集盘点 | 6-9 张（每个要点一张） |

### 6.2 图片比例

- 竖版 3:4（推荐，沉浸感强）
- 横版 16:9（适合对比图）
- 方形 1:1（适合产品特写）

### 6.3 图片风格

- 色调统一，不要超过 3 种主色
- 保持真实感，避免过度美化
- 文字说明清晰可读

## 七、自动化适配

### 7.1 小红书内容生成器

```javascript
// scripts/xiaohongshu-adaptor.js

/**
 * 将博客文章转换为小红书版本
 */
function generateXiaohongshuVersion(article) {
  // 1. 生成封面标题
  const coverTitle = generateCoverTitle(article.title);

  // 2. 精简正文内容
  const adaptedContent = simplifyContent(article.content, {
    maxLength: 500,
    preserveStructure: ['list', 'bold', 'image']
  });

  // 3. 生成话题标签
  const tags = generateWeiboTags(article);

  // 4. 生成互动引导
  const interactionPrompt = generateInteractionPrompt(article.category);

  return {
    title: coverTitle,
    content: adaptedContent,
    tags: tags.join(' '),
    interaction: interactionPrompt,
    coverImage: article.cover,
    checklist: [
      '封面图 3:4 比例检查',
      '标题字数检查（20字以内）',
      '正文字数检查（300-500字）',
      '标签数量检查（5-10个）',
      '互动引导检查'
    ]
  };
}
```

### 7.2 使用方法

```bash
# 生成单篇文章的小红书适配版本
node scripts/xiaohongshu-adaptor.js --file source/_posts/article.md

# 输出到 reports/xiaohongshu-article.md
```
