## 二、长文格式适配

### 2.1 知乎文章特点

| 特点 | 说明 |
|------|------|
| 篇幅 | 适合 2000-5000 字深度文章 |
| 结构 | 层次分明，小标题丰富 |
| 引用 | 支持丰富的引用和注释 |
| 代码 | 支持代码块（优于微信公众号） |

### 2.2 内容保留度

知乎专栏可以最大程度保留博客原文，适合技术类深度文章：

| 内容类型 | 保留度 | 说明 |
|---------|-------|------|
| 技术教程 | 95% | 代码块完美支持 |
| 深度分析 | 90% | 保留完整论述 |
| 观点解读 | 85% | 可适当精简 |
| 资讯速递 | 70% | 需要补充背景 |

### 2.3 格式转换规则

```javascript
// 知乎专栏格式转换
function adaptForZhihu(htmlContent) {
  return htmlContent
    // 知乎支持代码块，直接保留
    .replace(/<pre>/g, '```\n')
    .replace(/<\/pre>/g, '\n```')
    // 图片添加描述
    .replace(/<img/g, '<img alt="配图"')
    // 保留加粗、斜体等格式
    .replace(/<strong>/g, '**')
    .replace(/<\/strong>/g, '**')
    .replace(/<em>/g, '*')
    .replace(/<\/em>/g, '*')
    // 知乎支持引用块
    .replace(/<blockquote>/g, '> ');
}
```

## 三、排版规范

### 3.1 文章结构模板

```markdown
## 开头引导（100-200字）

[开门见山引入主题]
[简要说明文章价值和目标读者]
[如果需要，可添加目录导航]

---

## 一级标题（核心章节）

### 二级标题（细分内容）

正文内容...

### 二级标题

正文内容...

---

## 总结

[核心观点回顾]
[延伸阅读推荐]
[互动引导]

---

## 参考资料

- [参考资料1]
- [参考资料2]
```

### 3.2 排版要素

| 要素 | 建议 |
|------|------|
| 字数 | 建议 2000-5000 字 |
| 段落长度 | 每段 3-5 句话 |
| 小标题 | 每 500-800 字设置一个二级标题 |
| 重点强调 | 使用 **加粗**，避免过多斜体 |
| 引用 | 重要引用使用 `> 引用` 格式 |

### 3.3 代码块规范

```markdown
// 知乎支持的代码块格式
```javascript
function hello() {
  console.log('Hello, Zhihu!');
}
\`\`\`

// 带行号的代码块
\`\`\`python:hello.py
1 | def main():
2 |     print("Hello")
3 |
4 | if __name__ == "__main__":
5 |     main()
\`\`\`
```

## 七、SEO 与分发

### 7.1 标题优化

| 原则 | 说明 |
|------|------|
| 关键词前置 | 核心关键词放在前 15 个字 |
| 避免标题党 | 内容决定标题，标题服务于内容 |
| 长度控制 | 建议 20-40 字 |

### 7.2 标签策略

添加 3-5 个相关标签：

```javascript
// 知乎文章标签生成
function generateZhihuTags(article) {
  const tags = [];

  // 领域标签
  tags.push('程序员', '编程', '科技');

  // 从文章分类提取
  if (article.categories) {
    article.categories.slice(0, 2).forEach(cat => {
      tags.push(cat);
    });
  }

  // 从标签转换
  if (article.tags) {
    article.tags.slice(0, 3).forEach(tag => {
      tags.push(tag);
    });
  }

  return [...new Set(tags)].slice(0, 5);
}
```

## 八、自动化适配

### 8.1 知乎专栏适配器

```javascript
// scripts/zhihu-adaptor.js

/**
 * 生成知乎专栏版本
 */
function generateZhihuVersion(article) {
  return {
    title: article.title,
    content: adaptForZhihu(article.content),
    tags: generateZhihuTags(article),
    isOriginal: true,
    copyright: 'original',  // original: 原创, reprint: 转载
    footer: {
      source: '程序猿视界',
      url: article.url,
      authorization: true  // 是否允许规范转载
    },
    checklist: [
      '标题关键词检查',
      '正文结构检查',
      '代码块格式检查',
      '版权声明检查',
      '互动引导检查',
      '标签检查（3-5个）'
    ]
  };
}
```

### 8.2 使用方法

```bash
# 生成单篇文章的知乎适配版本
node scripts/zhihu-adaptor.js --file source/_posts/article.md

# 输出到 reports/zhihu-article.md
```
