# 社交媒体发布指南

**适用范围：** 文章发布到 shenhuanjie.github.io 后分享到社交媒体
**最后更新：** 2026-05-08

---

## 一、发布流程概览

```
文章发布 → 复制链接 → 选择平台 → 发布分享 → 记录数据
```

---

## 二、Twitter/X 发布指南

### 2.1 发布格式

**标题格式：**
```
[主题标签] 文章标题 | 关键亮点

🔗 链接
#标签1 #标签2 #标签3
```

**示例：**
```
🔥 2026年5月 AI资讯周报（第1周）

涵盖：OpenAI 发布 GPT-5、Claude 3.5 更新、
Anthropic 推出 AI 安全研究计划

🔗 https://shenhuanjie.github.io/2026/05/ai-weekly-01/
#AI #人工智能 #科技资讯 #OpenAI
```

### 2.2 发布规范

| 项目 | 要求 |
|------|------|
| 字符限制 | 280 字符以内 |
| 标签数量 | 2-5 个相关标签 |
| 配图 | 建议添加文章封面图 |
| @提及 | 如有相关账号可适当提及 |

### 2.3 发布时段建议

- 工作日：9:00-10:00、12:00-13:00、20:00-21:00
- 周末：10:00-12:00、15:00-17:00

## 三、LinkedIn 发布指南

### 3.1 发布格式

**标题格式：**
```
📖 文章标题

关键亮点/摘要...

🔗 阅读全文
#标签1 #标签2 #标签3
```

**示例：**
```
📖 Ollama 本地部署指南：快速搭建 AI 模型服务

本文介绍如何利用 Ollama 在本地快速部署和运行
大语言模型，无需云服务即可体验 AI 能力。

✅ 支持模型：Llama 2、Mistral、Code Llama
✅ 支持平台：macOS、Linux、Windows
✅ 适用场景：开发测试、隐私敏感环境

🔗 阅读全文：
https://shenhuanjie.github.io/2026/05/ollama-deploy-guide/

#人工智能 #开源大模型 #Ollama #开发者
```

### 3.2 发布规范

| 项目 | 要求 |
|------|------|
| 字符限制 | 3000 字符以内 |
| 摘要长度 | 100-200 字 |
| 标签数量 | 3-8 个相关标签 |
| 专业语气 | 保持专业、避免过度营销 |

### 3.3 LinkedIn 特殊技巧

- 在开头添加有价值的观点引发思考
- 使用换行提高可读性
- 避免纯链接分享，添加个人见解

## 四、知乎发布指南

### 4.1 发布格式

**标题格式：**
```
[相关问题邀请] 文章标题

文章摘要/核心观点...

原文链接：xxx
```

### 4.2 发布规范

- 选择相关问题进行回答
- 添加 1-2 张配图
- 结尾引导关注或讨论

## 五、微信朋友圈发布指南

### 5.1 发布格式

**标题格式：**
```
📚 文章标题

🔑 关键要点：
• 要点1
• 要点2
• 要点3

🔗 链接
#AI #科技 #学习
```

### 5.2 发布规范

- 保持简洁，3-5 个要点
- 使用 emoji 增加可读性
- 避免刷屏式分享

## 六、内容适配原则

### 6.1 各平台特点

| 平台 | 受众 | 风格 | 最佳发布时间 |
|------|------|------|--------------|
| Twitter | 开发者、科技爱好者 | 简短、即时 | 工作日午间 |
| LinkedIn | 专业人士、开发者 | 专业、深度 | 工作日早晨 |
| 知乎 | 知识追求者 | 专业、客观 | 晚间/周末 |
| 朋友圈 | 熟人圈 | 轻松、推荐 | 晚间 |

### 6.2 内容调整建议

| 原平台 | 目标平台 | 调整建议 |
|--------|----------|----------|
| 博客文章 | Twitter | 提炼 2-3 个核心观点 |
| 博客文章 | LinkedIn | 添加个人见解和实践价值 |
| 博客文章 | 朋友圈 | 简化为推荐理由+链接 |

## 七、自动化工具（可选）

### 7.1 Buffer

- 支持 Twitter、LinkedIn、Facebook
- 可预设发布时间
- 提供数据分析

### 7.2 Hootsuite

- 支持多平台统一管理
- 适合团队协作
- 提供内容日历

### 7.3 手动发布

由于博客更新频率不高，建议手动发布以保证内容质量和适配性。

## 八、发布检查清单

每次发布前确认：

- [ ] 文章已在博客正式发布
- [ ] 链接已复制并测试可用
- [ ] 标题和摘要已适配平台
- [ ] 添加了相关标签/话题
- [ ] 配图（如有）已添加
