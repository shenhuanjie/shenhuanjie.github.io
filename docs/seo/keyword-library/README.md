## 分类体系

### 一级分类（按领域）
- **AI大模型**：大模型技术、训练、推理
- **AI应用**：RAG、微调、Agent、提示词工程
- **智能化标准**：L1-L4、AI分级、政策法规
- **技术栈**：前端、后端、DevOps、架构
- **行业观察**：产业动态、投资融资、公司动态

### 二级分类（按搜索意图）
- **信息型**：教程、原理、概念解释
- **导航型**：资源导航、官网、工具
- **商业型**：评测、对比、选型
- **交易型**：注册、使用、配置

## 维护流程

1. **月度更新**：分析 Search Console 数据
2. **季度审视**：评估关键词效果
3. **年度重构**：重构关键词体系

## 贡献指南

添加新关键词时：
1. 确认关键词分类
2. 评估搜索量级和竞争度
3. 标注建议内容方向
4. 更新映射表
